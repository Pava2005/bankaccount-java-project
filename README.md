Bank Account Management System (Java)

Project Overview
This is a simple Bank Account Management System developed using Java. The project demonstrates Object-Oriented Programming (OOP) concepts such as Class, Object, Constructor, Methods, and Encapsulation.

Features
- Create a Bank Account
- Deposit Money
- Withdraw Money
- Check Account Balance
- Display Account Details
- Menu-Driven Program using Switch Case
- Input Handling using Scanner

Concepts Used
- Class and Object
- Constructor
- Methods
- Encapsulation
- Scanner
- Switch Statement
- Do-While Loop
- Conditional Statements

Technologies Used
- Java
- VS Code 

Program Flow
1. User enters account details.
2. Bank account object is created using a constructor.
3. Menu options are displayed.
4. User selects an operation:
   - Deposit
   - Withdraw
   - Check Balance
   - Account Details
   - Exit
5. The selected method is executed and the result is displayed.

Sample Menu
1. Deposit
2. Withdraw
3. Check Balance
4. Account Details
5. Exit
